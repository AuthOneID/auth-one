# AuthOne

**AuthOne** is a lightweight, self-hosted Identity Provider (IDP) built with modern web technologies. It provides secure authentication and authorization services for your applications, implementing industry-standard protocols like OpenID Connect (OIDC) and OAuth 2.0.

## ‚ú® Features

- **OpenID Connect (OIDC) & OAuth 2.0 Provider**: Securely authenticate users for your applications.
- **User Management**: Full control over users, roles, and groups.
- **Application Management**: Register and manage client applications.
- **Admin Dashboard**: Intuitive interface for administration.
- **API Access**: RESTful API for external integrations.
- **Modern Stack**: Built with AdonisJS, Inertia.js, and React.
